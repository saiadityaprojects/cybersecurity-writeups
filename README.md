# cybersecurity-writeups

This repository contains my personal writeups documenting hands-on practice in
penetration testing, network security, and web exploitation through various
lab environments and CTF-style challenges.

The goal of this repository is to demonstrate **methodology, reasoning, and
learning**, rather than just solutions.

---

## Platforms Covered

- **Hack The Box**
- **TryHackMe**

---

## Skills & Concepts Practiced

- Network and service enumeration
- Linux and Windows privilege escalation
- Web application exploitation
- SMB, FTP, HTTP, and MSSQL enumeration
- Basic Active Directory concepts
- Use of tools such as:
  - Nmap
  - Netcat
  - Gobuster
  - Metasploit (where applicable)
